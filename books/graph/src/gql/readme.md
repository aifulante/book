
- **GraphQL**

    API的查询语言，也是使用为数据定义的类型系统执行查询的服务器端运行时。
    
    GraphQL没有绑定到任何特定的数据库或存储引擎，而是由现有代码和数据支持。

    在Golang构建GraphQL服务器
